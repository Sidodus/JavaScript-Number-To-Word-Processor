JS Number To Word Processor v3.0.0
Is A JavaScript Plugin That Can Process Standard Dictionary Numbers Up To (1033003),
Decillinillion
Into Human Readable Numbers & Words
Based On Conway-Wechsler Numbering System

Change Decimal Length (default is 2 decimal place)

Number Would Only Be Called On Decimal Numbers.
If Number Is Greater Than The Decimal Point Length,
OR
Number Falls Outside (Range Numbering) Limit,
The Default 2 Decimal Point Would Be Used.
